How application Works

1. Use `escape key` to use mouse. (warning) there is a time limit

2. The Main Menu Button is the Logo in the middle-upper-center which will bring up the Navigation Menu

3. The Settings Menu (Black box at bottom) is where you can edit the time in seconds

4. Use mouse to look around the webSpace. Click on buttons that apply.

5. Press Q key for FAQ and keyboard Shortcuts
